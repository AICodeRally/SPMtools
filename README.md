# SPMtools

Sales Performance Management tools and models for analyzing, optimizing, and managing sales compensation plans.

## Structure

This repository contains various SPM tools organized by functionality:

- **tools/** - Individual tool implementations
  - Each subfolder contains a specific SPM tool with its own code, documentation, and dependencies

## Getting Started

Each tool in the `tools/` directory has its own README with specific setup and usage instructions.

## Contributing
